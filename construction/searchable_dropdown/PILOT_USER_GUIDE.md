# Enhanced Dropdown Search - User Guide
## For Pilot Users

---

## What's New

We've enhanced the dropdown search in **Journal Entry**, **Sales Invoice**, and other forms. Now you can:

✅ **Search by code OR name** (Arabic or English)
✅ **Hover to preview** selections (blue highlight)
✅ **See selected items** with ✓ checkmark
✅ **Clear selection** with "— بلا —" option

---

## How to Use

### 1. Open a Form

Go to **Accounting > Journal Entry** (or Sales Invoice, etc.)

### 2. Add a Line

Click **Add Row** in the accounts table

### 3. Click the Account Field

Click in the **Account** column

### 4. Start Typing

Type any of:
- **Account code**: e.g., "1000"
- **English name**: e.g., "Cash"
- **Arabic name**: e.g., "نقدية"

The dropdown will show matching results instantly.

### 5. Hover to Preview

Move your mouse over results:
- **Blue border** appears on the left
- Background tints light blue
- This helps you see which item is selected

### 6. Click to Select

Click the account you want. It will be selected and show:
- **Bold text**
- **✓ checkmark** on the right (or left in Arabic mode)

### 7. Clear If Needed

To remove the selection:
1. Open the dropdown again
2. Scroll to bottom
3. Click **"— بلا —"** (None)

---

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| ↓ (Arrow Down) | Move to next item |
| ↑ (Arrow Up) | Move to previous item |
| Enter | Select highlighted item |
| Escape | Close dropdown without selecting |

---

## Tips

💡 **Type partial names**: "bank" matches "Bank Account", "Bank Charges", etc.

💡 **Use Arabic**: If you know the Arabic name, type it directly

💡 **Hover first**: Preview with hover before clicking to confirm

💡 **Clear quickly**: Use "— بلا —" if you selected the wrong account
